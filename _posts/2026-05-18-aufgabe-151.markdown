Eine sehr angenehme Aufgabe. Man musste nur seine Zwischenergebnisse für die Erwartungswerte in einer Map "evals" speichern, um nicht für bereits berechnete Zustände wieder in die Rekursion zu gehen. 

Fangen wir mal damit an, die augenblickliche Anzahl unserer ganzen A1- bis A8-Blätter in einem Array "State" zu speichern (der große Vorteil ist, dass C++ diese Arrays automatisch lexikographisch ordnet, so dass wir beim Output nichts sortieren müssen). 

Abhängig von \\( N \\) sollen immer genau die hintersten \\( N \\) Stellen im Array den Zustand angeben (d.h. der \\( (8 - N) \\)-te Eintrag steht für die Anzahl der A1-Blätter). 
Um die Brüche für die Wahrscheinlichkeiten modulo \\( 10^9 + 7 \\) darzustellen, müssen wir zusätzlich noch modulare Inverse berechnen. Dazu adaptieren wir einfach die entsprechenden Funktionen aus Aufgabe 131 (und fügen zum Zeitsparen eine Map "invs" ein, um auch hier überflüssige Doppelberechnungen zu vermeiden). 

Hier die nötigen Vorbereitungen bis auf die entscheidende solve()-Funktion:

Fehlt also nur noch, dass wir ausgehend vom Startzustand mit genau einem A1-Blatt die Map "evals" füllen. Das ist aber nicht allzu schwierig. Wollen wir den Erwartungswert für einen beliebigen Zustand \\( s \\) berechnen, so sehen wir erstmal nach, ob \\( s \\) selbst die Bedingung erfüllt, dass genau **ein** Blatt im Umschlag liegt (die Anzahl an Blättern lässt sich ganz elegant mit "accumulate" berechnen). Je nachdem, starten wir mit einem Erwartungswert von 1 oder 0. Jetzt ziehen wir zufällig ein Blatt aus dem Umschlag (gehen also die einzelnen Elemente des Zustands durch), vollführen für dieses unsere "cut-in-half"-Prozedur (berechnen also den nächsten Zustand) und addieren dann den Erwartungswert für den nächsten Zustand (unter Beachtung eines Vorfaktors, der die WSK dafür angibt, dass wir überhaupt genau **dieses** Blatt aus dem Umschlag gezogen haben). Hier der Code mit Auslassungen:

{% highlight c++ %}

ull solve(const State s) {
    auto it = evals.find(s);
    if (it != evals.end())  return (it -> second);

    ull sheets = accumulate(s.begin(), s.end(), 0ULL);
    ull res = (sheets == 1);
